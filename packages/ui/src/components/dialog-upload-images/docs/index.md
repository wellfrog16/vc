# dialog-upload-images

## 功能

用于一次上传多张图片，支持**拖拽**排序

:::tip
此组件依赖 vue-draggable-plus，考虑此组件不一定会被用到，默认不安装 vue-draggable-plus。因此使用 tags 组件时需要

npm i vue-draggable-plus@^0.6.1
:::
