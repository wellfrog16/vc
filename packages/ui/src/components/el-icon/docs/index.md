# el-icon

## 功能

为了方便使用，全量加载 el-icon。

- 通过 `font-size` 控制 icon 大小，`color` 控制 icon 的颜色。
- 你可以通过 [element-plus](https://element-plus.org/zh-CN/component/icon#icon-collection) 挑选你需要 icon。
- 如果需要按需加载，请参考[官方教程](https://element-plus.org/zh-CN/component/icon#%E8%87%AA%E5%8A%A8%E5%AF%BC%E5%85%A5)
