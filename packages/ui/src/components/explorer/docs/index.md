# explorer

## 功能

管理后台常用布局功能组件的集合，包含以下组件：

- Explorer：根容器组件
- ExplorerPanel：面板容器组件
- ExplorerFilter：筛选区域组件
- ExplorerQuery：查询区域组件
- ExplorerTools：工具栏组件
- ExplorerList：列表区域组件
- ExplorerTable：表格区域组件
- ExplorerTree：树结构组件
- ExplorerFooter：底部组件（带分页）
