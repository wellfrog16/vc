# iconify-icon

## 功能

使用 [Iconify](https://iconify.design/) 图标库，提供图标组件。

依赖 [@iconify/vue](https://www.npmjs.com/package/@iconify/vue) ，使用 api 方式引入图标库，组件本身不包含图标。
