# tags

## 功能

基于 el-tag，用于标签组的编辑，可以对标签**双击**编辑、**拖拽**排序

:::tip
组件依赖 vue-draggable-plus，考虑此组件不一定会被用到，默认不安装 vue-draggable-plus。

pnpm add vue-draggable-plus@^0.6.1
:::
