# transfer

## 功能

穿梭框，用于多选。方便选择和展示。

比 [el-transfer](https://element-plus.org/zh-CN/component/transfer)，功能趋向于简洁和快捷

有两种方式：

- Transfer：穿梭器组件
- TransferPanel：穿梭器面板组件
