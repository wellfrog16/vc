## 功能

awesome-icon、el-icon、svg-icon 的三合一组件，消除了 3 种 icon 之间大小区别。

在牺牲加载速度的情况下，最大的限度的提供使用的便捷度。
- 通过 `font-size` 控制 icon 大小，`color` 控制 icon 的颜色。
